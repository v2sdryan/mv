MV Workflow

2 至 3 分鐘歌曲 -> 完整、有連貫性 MV

用 AI 做 MV，重點唔係「出片」，係先建立一條故事線。

呢個教學用 5 頁講清楚：先分析首歌，再寫 treatment、分鏡、角色與場景規則，之後用 ComfyUI 生成短片段，最後剪成同音樂同步嘅完整 MV。

流程地圖

最短可行流程

1. 聽歌拆段

Intro / Verse / Chorus / Bridge / Outro，每段一句情緒同畫面。

2. 寫 MV treatment

一句核心故事、主角、場景、顏色、結尾變化。

3. 生成 6 至 8 秒 shots

每段音樂用數個短鏡頭，先求一致，再求華麗。

4. 剪接同同步

用原曲做主軸，影片只係跟節奏、歌詞同情緒服務。

香港使用提示： 你用唔到 OpenAI API 都冇問題。文字策劃可以用 OpenRouter；影像生成主要靠本機、租 GPU、 RunComfy/ThinkDiffusion 類雲端 ComfyUI，或者其他你 VPN 後可用嘅服務。

3 個 Workflow 入口

由易到難，揀一條先完成 30 秒 demo。

GitHub 113k+ stars

Workflow A：ComfyUI Template -> 逐 shot 生成 MV

用 ComfyUI 做主工作台，載入官方 example/template workflow，先做 6 至 8 秒 image-to-video shot。適合新手， 因為所有節點、參數、輸出都可以睇到。

  1. 用 OpenRouter 寫 12 至 18 個 shots。
  2. 每個 shot 先生成一張 keyframe。
  3. 把 keyframe 丟入 ComfyUI I2V workflow。
  4. 用剪片軟件按歌曲 marker 排好。
GitHub 10.2k+ stars

Workflow B：LTX-Video 快速打草稿

用 LTX-Video 或 ComfyUI-LTXVideo 快速做 text-to-video / image-to-video 初稿。適合先確認故事節奏同鏡頭方向。

  1. 先輸出低解像度 preview。
  2. 每段歌詞試 2 至 3 個 motion prompt。
  3. 保留最順嘅版本，再重做重點鏡頭。
  4. Final chorus 可轉去 Wan 做高質版本。
GitHub 16k+ stars

Workflow C：Wan2.1 + Audio Reactive 做正式版

用 Wan2.1 或 WanVideoWrapper 做 cinematic shots，再加 Yvann Audio Reactive Nodes 令 chorus 跟鼓同 bass 反應。 適合做正式 MV 或重點片段。

  1. 固定角色、服裝、場景同色彩 prompt。
  2. 用 first frame / last frame 控制轉場。
  3. 把鼓、bass、人聲分開控制 1 至 2 個視覺參數。
  4. 最後用 VideoHelperSuite 輸出 MP4，再入剪片軟件。

Page 3

先做「MV Bible」，再生成畫面。

一條 MV 連貫，通常唔係因為每個 shot 都靚，而係每個 shot 都遵守同一套規則。

你要先準備

  • 音樂檔：WAV/MP3，最好有歌詞。
  • BPM 同段落時間：例如 0:00 intro，0:18 verse。
  • 主角設定：髮型、衣著、年紀、氣質。
  • 固定場景：最多 3 個，新手唔好太散。
  • 風格參考：電影、MV、攝影語言、色彩。

Page 4

生成策略：一段歌拆成好多短鏡頭。

唔好一次生成 3 分鐘。用 6 至 8 秒做單位，容易重做、容易對拍、容易保持質素。

01

先出角色同場景 keyframes

用 SDXL/Flux/Qwen Image 等先出靜態圖。每個角色保留 3 至 5 張最好版本，之後所有 I2V 都用佢哋做 reference。

02

用 LTX 快速試 shot

先做 480p 或較短片段。重點係鏡頭方向、構圖同動作節奏，唔好一開始追 4K。

03

用 Wan 做重點鏡頭

Chorus、開場、結尾可以用 Wan I2V/FLF2V 做更穩定嘅轉場同 cinematic motion。

04

用音訊反應做節奏層

鼓控制閃光或 camera shake，bass 控制 zoom，人聲控制粒子或顏色變化。唔好全部參數一齊郁。

一個實用生成配方

Shot 長度6-8 秒
FPS24 fps
初稿解像度480p 或 720p
每段候選3 個版本
剪接留白每 shot 頭尾多 0.5 秒
命名01_intro_a.mp4
版權同安全： 如果首歌唔係你自己擁有，公開發佈前要處理授權。下載 ComfyUI custom nodes 時只用可信 repo， 避免亂裝陌生節點。

Page 5

剪接、輸出、部署呢個教學網站。

最後一步係令所有片段服從首歌。你可以用 DaVinci Resolve、CapCut、Premiere 或 Final Cut。

剪接檢查表

  • 先放原曲，鎖定音軌。
  • 在每個 verse/chorus/drop 加 marker。
  • 先剪故事，再加效果。
  • 同一場景保持同一 LUT 或調色方向。
  • 最後先做字幕、片名、credits。
  • 輸出 H.264 MP4，另存 project 檔。

Vercel 部署

  1. 建立 GitHub repo，名叫 mv。
  2. 將呢個 mv 資料夾內容 push 上 repo。
  3. 去 Vercel -> New Project -> Import GitHub repo。
  4. Framework Preset 選 Other / Static。
  5. Build Command 留空，Output Directory 留 .。
  6. Deploy 後，你會得到公開教學網址。
